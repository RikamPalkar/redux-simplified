## Update rule for this learning project

If Redux or TypeScript implementation changes in source code, update the matching article in this series.

- Store or slice shape changes: Article 2
- Typed hooks or Provider changes: Article 3
- Dispatch logic changes in components: Article 4
- Selector or derived-state flow changes: Article 5
- Concept/foundation changes: Article 1

This keeps learning docs aligned with real code and avoids one oversized guide.
